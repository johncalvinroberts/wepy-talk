wtf is WePY?
  • A component-based framework for building WeChat mini programs
  • Optimizations on the existing MP framework, such as Promises, Async Functions, etc.
  • Official support from Tencent
  • https://github.com/Tencent/wepy
why would I use WePY?
  • Robust component system
  • Write modern javascript
  • Utilize the power of Node JS
  • Leverage open source
  • Write better code
